LoCoBot - Mobile ROS Robot

The LoCoBot is a ROS research rover for mapping, navigation & manipulation (optional) that enables researchers, educators & students alike to focus on high level code development instead of hardware & building out lower level code. Development on the LoCoBot is simplified with open source software, full ROS-mapping & navigation packages & modular open source Python API that allows users to move the platform as well as manipulate in as few as 10 lines of code.
